Reference Overlay Help

What it does: The “Load Ref” button overlays a reference hypnogram on top of the currently selected model output for the visible time window. The console prints accuracy and Cohen’s kappa over overlapping (non-null) epochs.

Supported reference formats

  • PhysioNet Sleep-EDF: *-Hypnogram.edf (EDF+ annotations)
  • OpenNeuro / BIDS: *_events.tsv (prefers stage_hum if present; falls back to stage_ai)

How alignment works

  • Epoch length is assumed to be 30 seconds.
  • Overlay alignment uses the current view window start time to slice reference epochs.
  • If the reference events start at a different origin than the EDF, the overlay may appear shifted.

Common troubleshooting

  • No overlay visible: confirm the ref file parsed to a non-empty stage array (check console for unique stages).
  • Looks shifted: try comparing PSG EDF with PSG events (same acquisition) before cross-device comparisons.
  • HMM effects: compare with HMM on/off; smoothing changes predicted stages only.
